# Protein–Protein Docking Pipeline

## 📌 Description

This pipeline was developed to perform **protein–protein docking**, integrating the workflow from preprocessing structures generated by **CHARMM/VMOD** to obtaining clustered complexes ranked by binding energy.

The workflow automates structural selection, clustering, input file preparation, docking execution using **HEX**, and organization of the results for downstream analyses.

---
## ⚙️ Requirements

The software requirements and dependencies are available at:

[Requirements](https://github.com/Labbiofisbiocomp/protein-protein-docking/blob/main/requirements.md)

---

# 📖 About the Project

This pipeline was developed to:

- select conformations based on MMFP energy;
- perform independent clustering of ligands and receptors;
- execute protein–protein docking using HEX;
- rank docking solutions according to binding energy;
- generate representative structures organized by clusters.

## 🔄 Pipeline Overview

```text
CHARMM / VMOD structure generation
        ↓
MMFP filtering (structure selection)
        ↓
Ligand and receptor clustering (independently)
        ↓
Selection of representative structures
        ↓
Structure preparation (chain renaming / formatting)
        ↓
Protein–protein docking (HEX)
        ↓
Energy ranking of docking solutions
        ↓
Selection of the top-ranked complexes
        ↓
Final clustering of docking solutions
        ↓
Cluster organization and analysis
```

## 📥 Inputs

* Structures generated in **CHARMM** using **VMOD**
* Ligand and receptor **PDB** files
* Reference structures for clustering

---

## 📤 Outputs

* Energy ranking (`energy_sorted.txt`)
* Selected complexes (`primeiras_10/`)
* Final clustered results (`pos_docking/cluster/`)
* Representative structures for each cluster

## 🧪 File Structure

- `00_montagem_v2.sh` → Creates the pipeline directory structure.
- `01_seleciona-mmfp_v4.sh` → Selects structures based on MMFP energy.
- `02_preparaClustering_v3.sh` → Prepares structures for clustering.
- `03_rodarClustering_v1.sh` → Performs clustering using GROMACS.
- `04_seleciona_estrutura-cluster_v3.sh` → Selects representative structures from each cluster.
- `05_cluster-structure_v2.sh` → Organizes the selected structures.
- `06_renomeia_cadeia_v5.sh` → Renames and adjusts PDB chain identifiers.
- `07_hex_v5.sh` → Performs protein–protein docking using HEX.
- `08_ordena-energia_v3.sh` → Sorts docking solutions by energy.
- `09_1_copiar_pdbs_rank_v2.sh` → Selects the top-ranked complexes.
- `10_preparaClusteringfinal_v2.sh` → Prepares complexes for the final clustering step.
- `11_rodarClusteringfinal-parte1_v2.sh` → First stage of the final clustering.
- `12_rodarClusteringfinal-parte2_v2.sh` → Second stage of the final clustering.
- `13_separa-organiza-clusters-docking_v4.sh` → Organizes the final results into cluster-specific directories.

---

## ⚠️ Important Notes

* The scripts must be executed in the order presented above.
* Ensure that the input structures are correctly formatted before starting the pipeline.
* Depending on the number of structures processed, some steps may require significant computational time.
* Verify the intermediate results before proceeding to the subsequent steps.
